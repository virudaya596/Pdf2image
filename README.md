# PDF2Image
Convert PDF pages to high-quality PNG images directly in your browser. No uploads, no server-side processing. Everything runs locally for maximum privacy and speed.

ðŸ”— Live Site: https://pdf2png-snowy.vercel.app

---

## Features
- Convert PDF pages to PNG images in the browser
- No file upload to any server (100% client-side)
- Individual page download
- Download all pages as a ZIP
- Drag & Drop upload support
- Mobile-friendly responsive UI
- Fast rendering using PDF.js

---

## Tech Stack
| Component | Technology |
|----------|------------|
| Frontend | HTML, CSS, JavaScript |
| PDF Engine | PDF.js |
| ZIP Generator | JSZip |
| Hosting | Vercel (Static) |

---

## Privacy
PDF2Image processes all files **only in your browser**.  
No files are uploaded or stored on any server.  
Your data remains completely private and secure.
